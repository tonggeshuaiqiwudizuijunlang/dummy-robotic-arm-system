# IMU参数简介

以下是对 `data[0]` 到 `data[7]` 所代表的飞行姿态参数的简要说明：

- ​**`data[0]`：滚转角X低8位（RollL）​**  
  滚转角X的低8位数据，用于计算滚转角。

- ​**`data[1]`：滚转角X高8位（RollH）​**  
  滚转角X的高8位数据，与低8位结合计算滚转角。  
  计算公式：  
  `滚转角X = ((RollH << 8) | RollL) / 32768 * 180 (°)`

- ​**`data[2]`：俯仰角Y低8位（PitchL）​**  
  俯仰角Y的低8位数据，用于计算俯仰角。

- ​**`data[3]`：俯仰角Y高8位（PitchH）​**  
  俯仰角Y的高8位数据，与低8位结合计算俯仰角。  
  计算公式：  
  `俯仰角Y = ((PitchH << 8) | PitchL) / 32768 * 180 (°)`

- ​**`data[4]`：偏航角Z低8位（YawL）​**  
  偏航角Z的低8位数据，用于计算偏航角。

- ​**`data[5]`：偏航角Z高8位（YawH）​**  
  偏航角Z的高8位数据，与低8位结合计算偏航角。  
  计算公式：  
  `偏航角Z = ((YawH << 8) | YawL) / 32768 * 180 (°)`

- ​**`data[6]`：保留位**  
  当前未使用，保留位。

- ​**`data[7]`：保留位**  
  当前未使用，保留位。

---

### 说明：
- 所有角度计算公式中，`<< 8` 表示将高8位数据左移8位，与低8位数据合并为16位数据。
- 计算结果单位为度（°）。
